Repo Prompt Configuration

Each repo's review prompt is configured via the ⚙ gear icon on its group header in the PR list. Current settings:

Bugs — Agent

Agent used when you run work against a bug.

Bugs — Repo Prompt Files

Pick a prompt file from each repo. When you start an agent on a bug and choose that repo, this file is appended to the work-item prompt.

Tickets — Agent

Agent used when you run work against a ticket (non-bug work item).

Tickets — Repo Prompt Files

Pick a prompt file from each repo. When you start an agent on a ticket and choose that repo, this file is appended to the work-item prompt.
