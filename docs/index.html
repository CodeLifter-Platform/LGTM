LGTM

A cross-platform menu bar app that lists your open Azure DevOps pull requests and runs AI-powered code reviews with a single click.

Features

Multi-Agent Support

Choose between Claude Code, Codex, and Augment Code, each with selectable models.

Clone & Review

Shallow-clones the repo using your PAT, runs the agent against the full codebase in isolation.

Streaming Output

Real-time markdown-rendered review results streamed directly into the app.

Per-Repo Prompts

Auto-detect from repo conventions, pick a file with autocomplete, or use the bundled default.

Secure PAT Storage

Your Azure DevOps token is stored in the OS keychain with encrypted fallback.

Live PR List

All active PRs grouped by repo, with starring, collapsible headers, and age filtering.

Auto-Review Queue

Review all open PRs sequentially with one click. Starred repos go first.

Webhook + Polling

Real-time updates via Azure DevOps Service Hooks with polling fallback.

Prerequisites

  • Node.js 18+ and npm
  • At least one AI agent CLI in PATH: claude, codex, or auggie
  • An Azure DevOps PAT with at least Code (Read) scope

Quick Start

npm install
npm start

On first launch the app appears in your menu bar / system tray and prompts for your Azure DevOps org URL and PAT.

Prompt Resolution

When starting a review, the prompt is resolved in this order: custom local path per-repo, a specific file within the repo (configured with autocomplete), convention auto-detect (.lgtm/review-prompt.md, .github/pr-review-prompt.md, PR_REVIEW_PROMPT.md), the global custom path, and finally the bundled REPO_REVIEW_TEMPLATE.md shipped with the app.
